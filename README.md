## 🧠 What is TabCost Pro?

TabCost Pro calculates how much money you lose every day due to forgotten, idle tabs. Based on your hourly rate, it accumulates a small cost per minute for each inactive tab (after a 5‑minute grace period). It’s a **metaphor for your attention** – not a real bill.

✅ Free version: daily cost, tab audit, close all inactive tabs, bilingual, dark mode.  
🚀 Pro version: ignore work domains, auto‑close, 365‑day history, CSV export, adjustable grace/impact, daily reset, smart notifications.

## 🔧 How to install and test locally (for developers)

1. Clone this repository.  
2. Open `chrome://extensions` in Chrome.  
3. Enable **Developer mode** (top right).  
4. Click **Load unpacked** and select the `extension/` folder.  
5. The extension icon will appear. Set your hourly rate in the popup.
