# Changelog

All notable changes to TabCost Pro will be documented in this file.

## [1.5.6] - 2026-06-12

- MV3 service worker with alarm-based cost engine (minuteTicker + heartbeat)
- Gumroad license verification with fingerprinting
- 365-day history chart (Pro feature)
- CSV export for daily history (Pro)
- Midnight rotation for daily cost
- Bilingual UI (English / Spanish)

### Changed
- Free tier now shows grace period selector (1 min or 5 min)

### Fixed
- Service worker termination with heartbeat fallback
- Cost calculation after Chrome restart
